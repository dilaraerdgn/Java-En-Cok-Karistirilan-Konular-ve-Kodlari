
# Java OOP ve Temel Kavramlar Rehberi

Bu depo, Java programlama dilinde öğrenilmesi kritik olan **Nesne Yönelimli Programlama (OOP)** prensiplerini ve temel dil yapılarını pratik örneklerle açıklamak amacıyla oluşturulmuştur. 

Öğrencilerin java öğrenim sürecinde en çok karıştırdığı konuları sade ve anlaşılır kod örnekleriyle bu dizin altında topladım. Yararlanmak isterseniz klosörleri inceleyebilirsiniz.

---

## İçerik ve Klasör Yapısı

Aşağıdaki tabloda, bu depoda yer alan temel konuların özeti bulunmaktadır:

| Klasör Adı | Açıklama |
| :--- | :--- |
| **AbstractClasses** | Soyut sınıfların kullanımı ve "is-a" ilişkisi örnekleri. |
| **Interfaces** | Arayüzlerin yetenek kazandırma (capability) ve çoklu kalıtım amaçlı kullanımı. |
| **Polymorphism** | Çok biçimlilik; aynı metodun farklı nesnelerde farklı davranışı. |
| **InnerClasses** | İç içe geçmiş sınıfların (Inner/Nested) kapsam ve erişim kuralları. |
| **Encapsulation** | Kapsülleme; verinin korunması ve Getter/Setter kullanımı. |

---

## Nasıl İncelenir?

1. İlgili konunun klasörüne gidin.
2. İçerisindeki `.java` dosyalarını açarak kod yapısını ve yorum satırlarını inceleyin.
3. Kendi IDE'nizde (IntelliJ, Eclipse, VS Code) projeyi klonlayarak kodları çalıştırabilirsiniz.
